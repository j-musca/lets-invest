lets-invest
===========

**Setup & Run**

* `mvn clean install`
* `mvn spring-boot:run`
    * comment in ´Xdebug -Xrunjdwp:transport=dt_socket,server=y,suspend=y,address=5005´ in pom.xml if you want run in debug mode.
